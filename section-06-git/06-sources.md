## The Essential Reference

- [**Pro Git** — free online book (git-scm.com)](https://git-scm.com/book/en/v2) — The official, comprehensive, and free Git book. Chapters 1–3 cover everything in this section. Chapters 5–6 cover collaborative workflows. This is the single best Git resource.

## Advanced Topics (When You're Ready)

- [Git Rebase — Atlassian](https://www.atlassian.com/git/tutorials/rewriting-history/git-rebase) — Clean linear history alternative to merging
- [Git Reset — Atlassian](https://www.atlassian.com/git/tutorials/undoing-changes/git-reset) — The powerful (and dangerous) history-rewriting tool
- [Git Hooks — Pro Git](https://git-scm.com/book/en/v2/Customizing-Git-Git-Hooks) — Run scripts automatically on git events
- [Git Tags — Pro Git](https://git-scm.com/book/en/v2/Git-Basics-Tagging) — Marking releases
- [Conventional Commits](https://www.conventionalcommits.org/) — A specification for structured commit messages
- [Git Flow — Atlassian](https://www.atlassian.com/git/tutorials/comparing-workflows/gitflow-workflow) — Workflow for release-based projects
