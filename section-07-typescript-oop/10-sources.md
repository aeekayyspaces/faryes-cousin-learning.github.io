## TypeScript Core

### Official Documentation
- [**TypeScript Handbook**](https://www.typescriptlang.org/docs/handbook/intro.html) — The official, comprehensive guide. Read it cover-to-cover when you're ready to go deep.
- [TypeScript from Scratch](https://www.typescriptlang.org/docs/handbook/typescript-from-scratch.html) — Start here if new to TypeScript
- [TypeScript for JavaScript Programmers](https://www.typescriptlang.org/docs/handbook/typescript-in-5-minutes.html) — Quick overview for JS developers
- [TypeScript Playground](https://www.typescriptlang.org/play) — Write and run TypeScript in the browser

### Type System
- [Everyday Types — Handbook](https://www.typescriptlang.org/docs/handbook/2/everyday-types.html)
- [Narrowing — Handbook](https://www.typescriptlang.org/docs/handbook/2/narrowing.html) — How TypeScript eliminates type possibilities
- [Generics — Handbook](https://www.typescriptlang.org/docs/handbook/2/generics.html)
- [Utility Types — Handbook](https://www.typescriptlang.org/docs/handbook/utility-types.html) — `Partial<T>`, `Readonly<T>`, `Pick<T>`, `Omit<T>` and more

### Articles
- [The TypeScript Tax — Eric Elliott](https://medium.com/javascript-scene/the-typescript-tax-132ff4cb175b) — Balanced counterpoint (worth reading)
- [TypeScript vs JavaScript — LogRocket](https://blog.logrocket.com/typescript-vs-javascript/)
- [Understanding TypeScript's strict mode](https://www.typescriptlang.org/tsconfig#strict)

## Books

| Book | Author | Notes |
|------|--------|-------|
| *Programming TypeScript* | Boris Cherny | The best dedicated TypeScript book; thorough and practical |
