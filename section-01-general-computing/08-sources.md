## The Command Line

### Interactive Learning
- [The Command Line Crash Course — Learn Code the Hard Way](https://learncodethehardway.org/command-line/) — Hands-on exercises
- [Bash Reference Manual — GNU](https://www.gnu.org/software/bash/manual/bash.html) — The definitive bash reference
- [explainshell.com](https://explainshell.com/) — Paste any shell command and get a plain-English explanation of every flag
- [tldr.sh](https://tldr.sh/) — Simplified man pages with practical examples

## Interactive and Visual Learning

These tools let you see computer science concepts in action:

- [Crash Course Computer Science (YouTube Playlist)](https://www.youtube.com/playlist?list=PL8dPuuaLjXtNlUrzyH5r6jN9ulIgZBpdo) — 40-episode series covering computing from transistors to AI. Highly recommended.
- [Computer Science Visualizations — USFCA](https://www.cs.usfca.edu/~galles/visualization/) — Visual animations of data structures and algorithms
- [How Computers Work — Khan Academy](https://www.khanacademy.org/computing/computers-and-the-internet/xcae6f4a7ff015e7d:computers) — Free interactive course
- [nand2tetris](https://www.nand2tetris.org/) — Build a computer from logic gates up. Advanced but incredible learning experience.

## Books

| Book | Author | Notes |
|------|--------|-------|
| *The Linux Command Line* | William Shotts | Free online; best command line book for beginners |
| *Operating System Concepts* | Silberschatz et al. | Standard university OS textbook |
| *Code: The Hidden Language of Computer Hardware and Software* | Charles Petzold | Builds a computer from scratch conceptually; very accessible |
| *The Innovators* | Walter Isaacson | Narrative history of the people who built computing |
| *Soul of a New Machine* | Tracy Kidder | Classic story of building a minicomputer in the 1970s |
| *Computer Organization and Design* | Patterson & Hennessy | CPU and hardware architecture deep dive |
